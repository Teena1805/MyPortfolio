Blood Group Detection Using Fingerprint

This project is a deep learning-based web application that analyzes movie reviews to predict whether the sentiment is positive or negative. It uses an LSTM model trained on 50,000 IMDB reviews. The system includes a Flask backend, interactive HTML frontend, and achieves over 87% accuracy in sentiment classification.

Main Highlights:

  • Predicts movie review sentiment as Positive or Negative using deep learning.
  • Trained on 50,000 IMDB reviews for high accuracy.
  • Uses LSTM model for better understanding of sequential text data.
  • Achieves 87.13% accuracy and 93.85% ROC AUC.
  • Built with Flask backend and HTML frontend.
  • Supports real-time prediction through a simple web form.
  • Easy to run locally and extendable for cloud deployment.
