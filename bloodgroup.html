Blood Group Detection Using Fingerprint

This project presents a novel approach to predicting a person’s blood group using fingerprint images and deep learning. By leveraging biometric data, the system eliminates the need for invasive blood tests. It uses a pre-trained VGG16 Convolutional Neural Network (CNN) model combined with advanced fingerprint preprocessing techniques such as CLAHE, noise reduction, and thresholding. The model is trained on a labeled dataset of fingerprint images categorized by blood group. A Flask-based web interface allows users to upload fingerprint images and receive real-time predictions with confidence scores, offering a fast, non-invasive, and intelligent health solution.

Main Highlights:

  • Detects blood group from fingerprint using deep learning.
  • Uses VGG16 CNN model with transfer learning.
  • Fingerprint preprocessing: CLAHE, noise removal, thresholding.
  • Dataset organized by blood group for training and testing.
  • Accuracy: ~86% training, ~82% validation.
  • Flask-based web app with user-friendly image upload.
  • Outputs predicted blood group with confidence score.
  • Includes SQLite database for saving user image info.
