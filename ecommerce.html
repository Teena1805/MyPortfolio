Ecommerce Website

This eCommerce Website is a complete online shopping solution featuring both user and admin interfaces. The user side allows customers to browse products, view details, add items to the cart, and place orders with a clean and responsive interface. On the admin side, authorized users can manage products—adding them individually or in bulk via Excel—view orders, and track inventory using a secure dashboard. The system is developed using HTML, CSS, JavaScript for the frontend and PHP with MySQL for backend operations. This project ensures a smooth shopping experience for users and efficient product and order management for admins.

  • User-friendly shopping interface with cart and order system.
  • Secure admin login to manage product and order data.
  • Single and bulk product upload via Excel (PHPSpreadsheet).
  • Product listing, search, filter, and category-based navigation.
  • MySQL database integration for user and product data.
  • Responsive layout for mobile and desktop users.
  • Real-time product and order updates for admin dashboard.
